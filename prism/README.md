## Usage

Inject following code to your footer.

```html
<script type="text/javascript"
        id="load-prism"
        src="https://ogkj5pyeh.qnssl.com/prism.js"
        data-prism-theme="twilight"
        data-prism-components="cpp python bash"
        data-prism-plugins="line-numbers"
        async>
</script>
```
